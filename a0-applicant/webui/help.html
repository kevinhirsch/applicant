Help & Instructions

Select a surface below for quick guidance.

Today

Your daily at-a-glance dashboard. Shows:

  • Upcoming deadlines and overdue items
  • Recent activity from your applications
  • Quick-action shortcuts for common tasks
Digest

Curated summary of notable updates. Use it to:

  • Review daily/weekly digest of changes
  • Filter by surface or time period
  • Mark items as read or actionable
Documents

Manage and view application-related documents. Supports:

  • Upload, preview, and download files
  • Organize documents by category
  • Search within document contents
Campaigns

Create and monitor outreach campaigns. Features:

  • Campaign templates and scheduling
  • Recipient list management
  • Performance tracking and analytics
Activity

Real-time log of all actions across surfaces. Use to:

  • Track changes and user interactions
  • Filter by surface, date, or user
  • Export activity logs for auditing
Chat

AI-assisted messaging interface. You can:

  • Ask questions about your applications
  • Request document generation or summarisation
  • Issue commands to automate workflows
Live session

Active real-time collaboration sessions. Provides:

  • View current live session participants
  • Share screen or documents in real time
  • Session recording and playback controls
Health

System status and diagnostics. Monitor:

  • API connectivity and service uptime
  • Recent errors and warnings
  • Performance metrics and resource usage
