Keyboard Shortcuts

Shell Shortcuts
Close any open modal or panel Esc
Save or approve a pending action Ctrl + S
Pass or reject the current item Ctrl + Shift + P
Search or filter content Ctrl + F
Panel Navigation
Open a panel by clicking its sidebar button Click sidebar buttons
Navigate between open panels Click sidebar buttons
Close the current panel or modal Esc
Common Panel Actions
Save or approve changes Ctrl + S
Pass or reject current selection Ctrl + Shift + P
Search within the active panel Ctrl + F
Toggle filter options Ctrl + Shift + F
Surface Tips
  • Click the ? button in any panel title for context help.
  • Use sidebar buttons to switch between surfaces.
  • Press Esc to close modals and return to the main view.
